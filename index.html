Scribe Rich Text Editor

Scribe is an open source editor build for the modern web. It is built with an extensible architecture with an expressive API so you can completely customize it for your use case. Some built in features include:

- Fast and lightweight
- Semantic markup (copy and paste with other editors)
- Standardized HTML between browsers
- Supports Chrome, Firefox, Safari, and IE 8+

Full Featured API

Get fine grain access to editor contents and event notifications.

Cross Platform

Works across all modern browsers on desktops, tablets and phones.

Customizable

It's easy to add custom behavior or modifications on top of Scribe.

Scribe is open source. It is hosted and maintained on GitHub.
